# Robopupu-FRP

<img src="https://github.com/Fuusio/Robopupu/blob/gh-pages/images/robopupu_header_image.png" alt="Robopupu mascot"/>

A simple Android library for [Functional Reactive Programming FRP](https://en.wikipedia.org/wiki/Functional_reactive_programming).

Check out the [Robopupu project website](http://robopupu.com/robopupu-frp/) for further information.

## Documentation

* [Robopupu developer documentation](http://robopupu.com/robopupu-frp/)
* [Javadoc API documentation](http://fuusio.github.io/Robopupu-FRP/javadocs/)
